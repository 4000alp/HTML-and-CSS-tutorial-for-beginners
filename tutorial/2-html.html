What is HTML?

Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications. With Cascading Style Sheets (CSS) and JavaScript, it forms a triad of cornerstone technologies for the World Wide Web.

In other words - each and every webpage which you see on the web uses HTML. Let's check it. <p> and <div> looks simmilar but they have semantic difference - a <div> element is designed to describe a container of data whereas a <p> element is designed to describe a paragraph of content. If you need a generic container purely for layout purposes then use a <div> . If you need an element to describe a paragraph of content then use a <p>

Tags

<body>, <head>, <h1>, <p>, <div> - these are all HTML elements called tags.

An HTML element usually consists of a start tag and end tag, with the content inserted in


content goes here...
like we've seen before: 

<p>Some paragraph, with some text </p>

however, there are singular tags which don't require closing tags i.e.:


<br/>
<img src="myCat.jpg" alt="Image showing cat" />

Note: For the sake of this tutorial, it's sufficient knowledge, however, in order to really learn HTML and CSS you should learn more about tags later.

Nested HTML Elements

As you can see base on our current code - <h1>, <p>, and <div> tags are nested under <body> tag, which is nested under <html> tag which is a top tag. Netsing allows us to group elements and make code more readable.

Attributes

Expect for the fact that we can put some data in between tags, they (tags) can also have attributes, like <img> tag which you have already seen.

  1. All HTML elements can have attributes.
  2. Attributes provide additional information about an element.
  3. Attributes are always specified in the start tag

Example:


<img src="cat.jpg" width="500" height="600">

One of the most commonly used attribute is a class attribute which allows us to apply CSS styles to our HTML elements.
