Indents

As you noticed, I am using intends to organize my code. Intends (usually 2 or 4 spaces or 1-2 tabulators) help us to understand nested code.

Links

We have learned how to connect external CSS file. Now let's learn how to link other files to our website.

In order to connect our index.html and project.html page we will use new tag. Let's type the following:


<a href="project.html">Go to projects</a>

When you click it you will be redirected to our project.html page. Since we don't have link on project.html file to index.hml we have to click "Return" in our browser or add return link in our project.html file:


<a href="index.html">Back to main page.</a>

The <a></a> tag is commonly used. We can use it redirect user to:


<a href="http://mdbootstrap.com/education">More tutorials</a>

We can also use it to display full size images.


<a href="https://mdbootstrap.com/img/logo/mdb-transparent-big.png">Enlarge image</a>

Or even attach files PDF or archived (zip/rar) files to download:


<a href="https://mdbootstrap.com/img/logo/mdb-transparent-big.pdf">Download PDF</a>

Note: some browsers instead of download may open PDF file.


<a href="https://mdbootstrap.com/img/logo/mdb-transparent.zip">Download .zip file</a>

Not only text

So far, we used text for hyperlinks almost anything can be used as a hyperlink - i.e. image:


<a href="https://mdbootstrap.com/img/logo/mdb-transparent-big.png"><img src="https://mdbootstrap.com/img/logo/mdb192x192.jpg"></a>

In our project, we will use button to redirect user to project.html website.


<a href="project.html">
    <button class="register">JOIN NOW</button>
</a>
